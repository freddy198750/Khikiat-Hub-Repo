## ЁЯЗмЁЯЗз About This Project (English)

**Khikiat Hub** is a GUI-based AutoHotkey launcher I created for fun and self-learning.  
It allows users to select a game and script function, then dynamically fetches and runs that AHK script directly from GitHub тАУ no pre-downloads required.
